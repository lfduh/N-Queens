# N-Queens
language : C#  
platform: windows

## Instructions

with [dotnet](https://dotnet.microsoft.com/en-us/download) installed

### build
batch script in
```
.\N-Queens\build.bat
```

result path:
```
.\N-Queens\N-Queens\bin\Debug\net6.0\N-Queens.exe
```

### run 
batch script in
```
.\N-Queens\run.bat
```
